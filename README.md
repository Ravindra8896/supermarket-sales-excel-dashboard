# supermarket-sales-excel-dashboard
Interactive Excel dashboard analyzing 1000+ supermarket sales transactions to uncover revenue trends, customer behavior, and product performance insights.
📊 Supermarket Sales Dashboard (Excel)
🔍 Project Overview

This project analyzes 1000+ supermarket transactional records using Microsoft Excel to uncover business insights related to revenue trends, customer behavior, product performance, and payment preferences.

The objective was to transform raw sales data into an interactive and business-driven dashboard.

🛠 Tools & Techniques Used

Microsoft Excel

Pivot Tables

KPI Cards

Slicers (Interactive Filters)

Data Cleaning & Formatting

Derived Time Features (Month, Day, Hour)

Data Visualization & Dashboard Design

📈 Key Performance Indicators (KPIs)

💰 Total Revenue: $322,966.75

📦 Total Orders: 1000

⭐ Average Customer Rating: 6.97

🛒 Average Order Quantity: 5.51

📊 Dashboard Insights

🥇 Food & Beverages is the highest revenue-generating product category.

👩 Female customers contribute approximately 60% of total sales.

💳 E-wallet is the most preferred payment method.

📅 January recorded the highest monthly revenue.
